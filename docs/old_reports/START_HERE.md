## 📞 TROUBLESHOOTING

### **Issue: "Cannot connect to MongoDB"**
```
Solution:
1. Check MongoDB running: Get-Service MongoDB
2. Verify connection string in .env
3. Restart backend: Ctrl+C, npm start
```

### **Issue: "Port 5000 already in use"**
```
Solution:
1. Kill processes: Get-Process -Name node | Stop-Process -Force
2. Restart backend: npm start
3. Or change PORT in .env and restart
```

### **Issue: "Frontend won't load"**
```
Solution:
1. Check backend running: http://localhost:5000/api/health
2. Clear browser cache: Ctrl+Shift+Delete
3. Restart frontend: Ctrl+C, npm start
4. Check console errors: F12 in browser
```

### **Issue: "API requests fail"**
```
Solution:
1. Verify backend on port 5000
2. Check proxy in client/package.json
3. Review API endpoint URLs
4. Restart both servers
```
